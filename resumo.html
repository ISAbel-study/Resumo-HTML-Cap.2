Resumo: Capítulo 2 - Estruturação de Páginas Web com HTML

1. Introdução e Fundamentos

  • Papel do HTML: Utilizado para definir a estrutura e a semântica dos componentes de uma página web.
  • Versões:
    • HTML: A primeira versão (sem número de versão) foi publicada em 1992
    • HTML 4.0: Lançado em 1997 e importante porque o CSS se tornou disponível nesta versão.
    • XHTML 1.0: É a 'junção' entre XML e HTML, e possui regras mais restritas
    • HTML5: Esta versão foi lançada em 2014 e é escrito com o número da versão logo após “HTML”
  • Elementos e Atributos:
    • Elementos: Tags de abertura e fechamento que delimitam conteúdo e o corpo da página.
    • Atributos: Para adicionar configurações para os elementos.
  • Documentos/arquivos HTML: Devem ter a tag <!DOCTYPE html>, para informar ao navegador qual versão html está sendo usada.

2. Formatação de Texto e Estruturas Básicas

  • Títulos e Parágrafos: Uso de <h1> a <h6> para hierarquia dos títulos e <p> para blocos de texto, parágrafos.
  • Destaques: Negrito (usando <strong> ou <b>) e Itálico (usando <em> ou <i>).
  • Outros: sobrescrito, subscrito e quebra de linha manual com <br />.
  • Citações: <blockquote> para blocos longos e <abbr> para abreviações.

3. Listas, Links e Mídia

Listas Ordenadas:
  1. Entradas numeradas sequencialmente.
  2. Tag (<ol>) para criar a lista ordenada.
  3. Tag (<li>) para criar os itens da lista.
Listas Não Ordenadas:
  • Entradas com marcadores (bullets).
  • Tag (<ul>) para criar a lista não ordenada.
  • Tag (<li>) para criar os itens da lista.
Listas de Definição
Tag (<dl>) para criar a lista de definição.
Na lista de definição para criar os itens da lista, no lugar do elemento (<li>) usa-se os pares de elementos (<dt>) e (</dt>), (<dd>) e (</dd>).
Links
Navegação via URLs externas, caminhos relativos ou âncoras internas.
Tag (<a>) com o atributo (<href>) para apontar para outra página.
Existem vários tipos de links: Links externos levam você a outros sites, links relativos pulam para outra página da web no mesmo site, e links internos vão para outro local dentro da mesma página da web.
Imagens
Tag (<img>) para inserir imagens.
Usa o atributo src para especificar a URL onde a imagem está.
Usa o atributo alt para colocar um texto alternativo para a imagem caso ela for exibida no navegador.
Os elementos (<figure>) e (<figcaption>) são para adicionar uma legenda a uma imagem.

4. Tabelas e Formulários

Tabelas

Elemento Função
<table> Definir a estrutura de uma tabela em HTML
<td>,<tr> Inserir novas linhas na tabela com o elemento <tr> e novos dados pelo o elemento <td>
<thead>,<tbody>,<tfoot> Os elementos especiais <thead> (cabeçalho da tabela), <tbody> (corpo da tabela) e <tfoot> (rodapé da tabela) são usados para marcar a área da tabela.
Formulários
Tag <form> para definir o formulário.
Elementos <fieldset> opcionais permitem agrupar áreas individuais em um formulário.
Definir campos de texto, campos de senha, botões de rádio e caixas de seleção usando o <input>.
O atributo <type> define o tipo do campo de entrada: para campos de texto, é text; para campos de senha, password; para botões de rádio, radio; e para caixas de seleção, checkbox, endereços de e-mail, email.
Definir listas suspensas usando o <select>, definir as opções individuais dentro de uma lista suspensa usando <option> e definir áreas de texto usando <textarea>.
Adicionar um rótulo a um campo de entrada, usar o elemento <label>.

5. Comportamento de Exibição

  • Bloco (Block): Ocupam toda a largura e iniciam nova linha (ex: <p>, <h1>).
  • Em linha (Inline): Mantêm-se no fluxo do texto (ex: <a>, <strong>).
